# UFW 防火墙优化脚本

仅允许中国 IP 访问指定端口的 UFW 防火墙规则管理工具。

## 功能特性

- 自动获取中国 IP 段 (`apnic.net`)
- 仅允许中国 IP 访问指定端口
- 自动检测内网端口并跳过限制
- 规则备份与恢复
- 错误自动回滚

## 快速开始

```bash
# 运行脚本 (需要 sudo)
sudo ./banIP.sh
```

选择操作：
- **1**: 备份当前规则，仅允许中国 IP 访问指定端口
- **2**: 恢复到备份的规则

## 运行时文件

脚本运行时会自动生成以下文件：

| 文件 | 说明 |
|------|------|
| `restricted_ports.txt` | 从当前 UFW 规则提取的端口 (自动去重) |
| `china_ips.txt` | 中国 IP 段列表 |
| `ufw.policy` | UFW 规则备份 |

这些文件可以在运行后删除，不影响再次运行。

## 注意事项

- 脚本需要 `sudo` 权限
- 运行前请确保了解防火墙配置
- 误配置可能导致无法远程访问
- 建议首次在测试环境运行
