# UFW 防火墙优化脚本

限制指定国家/地区 IP 访问的 UFW 防火墙规则管理工具。

## 功能特性

- 支持 8 个国家/地区选择 (中国、印度、俄罗斯、美国、日本、韩国、巴西、印度尼西亚)
- 自动获取选定区域的 IP 段 (`apnic.net`)
- 仅允许选定区域 IP 访问指定端口
- 自动检测内网端口并跳过限制
- 规则备份与恢复 (支持 ufw export/import)
- 错误自动回滚

## 支持区域

| 代码 | 国家/地区 |
|------|-----------|
| CN | 中国 |
| IN | 印度 |
| RU | 俄罗斯 |
| US | 美国 |
| JP | 日本 |
| KR | 韩国 |
| BR | 巴西 |
| ID | 印度尼西亚 |

## 快速开始

```bash
# 运行脚本 (需要 sudo)
sudo ./banIP.sh
```

### 操作流程

1. **选择操作**: 设置防火墙规则 或 恢复原有规则
2. **选择区域**: 从支持的 8 个国家/地区中选择
3. **确认**: 确认后将只允许该区域 IP 访问

## 运行时文件

脚本运行时会自动生成以下文件：

| 文件 | 说明 |
|------|------|
| `restricted_ports.txt` | 从当前 UFW 规则提取的端口 (自动去重) |
| `china_ips.txt` | 选定区域的 IP 段列表 |
| `ufw.policy` | UFW 规则备份 |

这些文件可以在运行后删除，不影响再次运行。

## 注意事项

- 脚本需要 `sudo` 权限
- 运行前请确保了解防火墙配置
- 误配置可能导致无法远程访问
- 建议首次在测试环境运行
